Client:

Contains an html file. This file is to be opened on the client computer connecting to the raspberry PI.
You will need to enter the IP address of the PI you are connecting to in order to download the switch information.

Server:

The server iss the raspberry pi. The pi should be connected to GPIO's to control.
You will need to know the IP address of the PI to connect to it, and the included python script must be running in orer to connect to it via http.
